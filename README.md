# Catching Game

![Preview](/public/img/preview.webp)

Catch all the falling treasures while avoiding the enemies walking across the game.

## Getting Started

First you will need to setup a .env file from the .env.sample provided.

Then development server can be run:

```bash
npm run dev
```

# Controls
✅ Keyboard Support
⚠️ Controller Support
✅ Touch Support (via joystick and click to move)
